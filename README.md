# 个人学术记录网站

简约学术风格的静态个人网站，用于记录和展示读研期间的每日收获与进展。

## 五大板块

| 板块 | 展示方式 |
|---|---|
| 个人信息 | 静态卡片：教育背景、技能、联系方式 |
| 读研生活 | 卡片列表：日常感悟、组会总结、读书笔记 |
| 项目进展 | 项目 → 时间线：点击项目查看完整进展 |
| 实验操作 | 实验 → 时间线：点击实验查看操作记录 |
| 论文写作 | 论文 → 时间线：点击论文查看写作进度 |

## 使用方法

### 本地查看
直接用浏览器打开 `index.html` 即可，无需服务器。

### 日常更新
只需编辑 `js/data.js` 文件：

**添加读研生活 / 文献笔记**：在对应数组最前面复制一条模板，修改日期、标题、内容。

**添加项目 / 实验 / 论文进展**：找到对应实体，在它的 `timeline` 数组最前面插入一条新记录。

### 部署上线（免费）

#### GitHub Pages
1. 创建 GitHub 仓库，命名为 `你的用户名.github.io`
2. 将所有文件上传到仓库
3. Settings → Pages → Source: main branch → Save
4. 几分钟后访问 `https://你的用户名.github.io`

#### 或 Netlify
1. 注册 netlify.com
2. 将 `D:\personal_web` 文件夹拖拽到 Netlify 页面
3. 自动生成网址
